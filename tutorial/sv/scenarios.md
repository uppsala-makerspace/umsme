# Existerande medlemmar
* Medlem skapar konto med samma email som finns i vårt register
* Medlem bekräftar email (ej om SSO med google, då behövs inte det)
* Medlem godkänner senaste versionen av ansvarsförbindelsen
* Medlem kan se sin medlemsstatus, betalningar, osv.
* Medlem kan öppna dörrar (om labbmedlemskap)

# Existerande medlem förnyar medlemskap
* Medlem får påminnelse om att medlemskap snart behöver förnyas (14 dagar innan via mail, mer ofta i appen via push notifieringar)
* Medlem väljer medlemskapsform i appen och initierar betalning via swish
* Betalning sker i swishappen och medlemmen återvänder till appen
* Medlem får bekräftelsemail på genomförd betalning
* Medlem kan se ny medlemsstatus samt betalningen i appen
* Medlem kan öppna dörrar (om labbmedlemskap)
